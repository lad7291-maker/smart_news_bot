# Рекомендации по улучшению SmartNews Bot

## На основе анализа 97 постов канала SmartNews

---

## Проблема: Абсурдные/нерелевантные фото

### Статистика:
- ~60% фото абсурдные/нерелевантные
- ~20% нейтральные (логотипы источников)
- ~20% релевантные (политики, события, графики)

### Примеры абсурдных фото:
| Пост | Фото |
|------|------|
| Трамп об ударе по иранской школе | Дети с сердечками |
| Трамп и Ким Чен Ын | Фабрика шоколада Lindt |
| Цены на нефть | Логотип YouTube |
| Президент Абхазии и Путин | Обложка клипа Freya Skye |
| Трамп обсудил с Си Цзиньпином | Математическая презентация |
| Аракчи о перемирии с США | Французский учебник |

---

## Внесенные улучшения

### 1. Новый модуль: image_relevance_checker.py
- Проверка релевантности фото ПЕРЕД отправкой
- Обнаружение абсурдных паттернов
- Score система: >=40 = релевантно

### 2. Улучшен image_search.py
- Проверка релевантности найденного фото
- Fallback на флаг/логотип если фото не подходит

### 3. Улучшен poster.py
- Поддержка fallback-изображений

### 4. Улучшен bot_runner.py
- Метки fallback при публикации

### 5. Расширен черный список _BLOCKED_TOPICS
Добавлены: vevo, music video, lindt, chocolate, methodologique, hodge, youtube

---

## Хороший паттерн оформления

```
[Флаг] [Краткий заголовок 6-10 слов]
[Релевантное фото]
[Главное/Кратко/Суть + 2-4 абзаца]
[Читать полностью + ссылка]
[#Источник #новости]
```
