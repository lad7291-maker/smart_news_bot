# Инструкция: Покупка домена на Hostland + настройка на сервере

## ШАГ 1: Регистрация/вход в Hostland

1. Откройте https://hostland.ru/
2. Нажмите "Войти" (если есть аккаунт) или "Регистрация" (если нет)
3. Авторизуйтесь

---

## ШАГ 2: Проверка доступности домена

1. На главной странице найдите поле "Проверить домен"
2. Введите: `smartnewsai.ru` (или другой желаемый)
3. Нажмите "Проверить"
4. Если домен свободен — нажмите "Зарегистрировать"

**Рекомендуемые домены (проверьте все):**
- `smartnewsai.ru` — основной для РФ
- `smartnewsai.com` — международный
- `smartnewsai.io` — технологичный
- `smartnewsai.tech` — бюджетный вариант

---

## ШАГ 3: Оформление заказа

1. Выберите период регистрации (1 год минимум)
2. В разделе "DNS-серверы" выберите: **"Использовать DNS-серверы Hostland"**
3. **НЕ покупайте** дополнительные услуги (SSL, хостинг, защита WHOIS) — мы настроим бесплатно
4. Нажмите "Оформить заказ"
5. Оплатите

---

## ШАГ 4: Настройка DNS (A-запись)

После покупки перейдите в раздел "Мои домены" → выберите ваш домен → "Управление DNS"

**Создайте/измените A-запись:**

| Тип | Имя | Значение | TTL |
|-----|-----|----------|-----|
| A | @ | 94.156.122.191 | 3600 |
| A | www | 94.156.122.191 | 3600 |

**Если есть CNAME для www — удалите его, замените на A-запись.**

Нажмите "Сохранить"

---

## ШАГ 5: Проверка DNS (подождите 15-30 минут)

Откройте терминал на вашем сервере и проверьте:

```bash
# Замените smartnewsai.ru на ваш домен
dig smartnewsai.ru +short
# Должно показать: 94.156.122.191

dig www.smartnewsai.ru +short
# Должно показать: 94.156.122.191
```

---

## ШАГ 6: Настройка Nginx на сервере (я сделаю это за вас)

После того как DNS обновится (обычно 15-60 минут), сообщите мне:
1. Какой домен купили
2. Я настрою Nginx + SSL

**Или выполните сами:**

```bash
# 1. Создайте конфиг для домена
sudo nano /etc/nginx/sites-available/smartnewsai
```

Вставьте:
```nginx
server {
    listen 80;
    listen [::]:80;
    server_name smartnewsai.ru www.smartnewsai.ru;

    location / {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

```bash
# 2. Активируйте конфиг
sudo ln -s /etc/nginx/sites-available/smartnewsai /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

# 3. Установите Certbot для SSL
sudo apt update
sudo apt install certbot python3-certbot-nginx

# 4. Получите SSL-сертификат
sudo certbot --nginx -d smartnewsai.ru -d www.smartnewsai.ru

# 5. Автоматическое обновление SSL
sudo systemctl enable certbot.timer
```

---

## ШАГ 7: Обновление лендинга

После настройки домена, обновите в лендинге:

1. В `index.html` замените все `https://smartnewsai.ru/` на ваш реальный домен
2. В `robots.txt` замените `https://smartnewsai.ru/sitemap.xml`
3. В `sitemap.xml` замените `<loc>https://smartnewsai.ru/</loc>`
